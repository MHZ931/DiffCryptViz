This is the first few rounds of the SKINNY tweakable block cipher. Designed to be lightweight.


Cipher Description (What the letters in the figure means):

SubCells (SC): An S-Box is applied to each cell

AddConstants (AC): A 6-bit affine LFSR, that generates the round constant

AddRoundTweakey (ART): Xor each state with a tweakey (Key with a tweakable schedule)

ShiftRows (SR): Shift the row cells, as specified in the figure above

MixColumns (MC): An operation that xors and reorganizes the rows, as specified in the figure above

Page Description:

Active cells: You can specify the active cells in the input box (integers 0-15, separated by comma), these numbers will effect the underlying TikZ code, which is then rendered by TikzJax, where each active cell is marked in red and subsequently changed by the cipher operations.

Rounds: Specify the number of rounds, ranging from 1 to 8. If you're using this for personal use, you can make the max number larger, but do expect long render time from TikzJax.
